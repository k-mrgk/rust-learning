## learning_3-2

フィボナッチ数列の n 番目を生成する。

```
$ cargo run
Outputs the n-th Fibonacci number
Input n > 0
0th Fibonacci number is 0


$ cargo run
Outputs the n-th Fibonacci number
Input n > 1
1th Fibonacci number is 1

$ cargo run
Outputs the n-th Fibonacci number
Input n > 50
50th Fibonacci number is 12586269025

```
